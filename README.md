# Estudo de TypeScript

Este repositório tem como objetivo reunir anotações, exemplos de código e resultados práticos dos meus estudos em TypeScript, servindo como um material de referência e acompanhamento da minha evolução na linguagem.

---

## Objetivo

- Organizar e documentar o aprendizado de TypeScript de forma estruturada e acessível.  
- Criar um histórico de progresso com anotações, exemplos de código e observações pessoais.  
- Montar um repositório que possa servir tanto para consulta futura quanto para compartilhamento com a comunidade.

---

## Recursos de Estudo

Os estudos estão sendo baseados nos seguintes vídeos:

- [Curso de TypeScript para completos iniciantes](https://www.youtube.com/watch?v=ppDsxbUNtNQ&pp=ygUtY3Vyc28gZGUgdHlwZXNjcmlwdCBwYXJhIGNvbXBsZXRvcyBpbmljaWFudGVz)
- [Playlist completa sobre TypeScript](https://www.youtube.com/watch?v=u7K1sdnCv5Y&list=PLb2HQ45KP0Wsk-p_0c6ImqBAEFEY-LU9H)

Esses materiais são utilizados como base para as anotações e exemplos que estarão organizados neste repositório.


---

## Metodologia de Estudo

1. Assistir ao vídeo correspondente ao módulo.  
2. Praticar os conceitos apresentados.  
3. Criar uma anotação em Markdown na pasta `Anotacoes/` com os principais pontos e exemplos.  
4. Implementar exemplos de código na pasta `Exemplos/`.  
5. Desenvolver pequenos projetos práticos na pasta `ProjetosPraticos/` aplicando o conteúdo estudado.  
6. Fazer revisões periódicas para reforçar o aprendizado.
